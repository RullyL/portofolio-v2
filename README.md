# Personal Portfolio (Next.js + TypeScript + Tailwind + Ant Design)

This scaffold provides a production-ready starting point matching the requested spec: App Router, strict TypeScript, Tailwind CSS, Ant Design, dark/light theme support, accessible markup, and reusable components.

Quick start

1. Install:

```bash
npm install
```

2. Run dev:

```bash
npm run dev
```

Notes
- Theme: `useTheme` persists preference to localStorage and respects `prefers-color-scheme`.
- Tailwind dark mode uses the `dark` class on `<html>`.
- Ant Design reset imported and token color wired to CSS variable `--color-primary`.
